☁ Configurar Sincronização em Nuvem
Firebase Realtime Database — gratuito, sem servidor
① Criar projeto no Firebase
  1. Acesse console.firebase.google.com (entre com conta Google)
  2. Clique em "Adicionar projeto"
  3. Digite um nome (ex: opec-master-tv) e clique Continuar
  4. Desative o Google Analytics (não precisa) → Criar projeto
  5. Aguarde criar e clique Continuar
② Ativar o banco de dados
  1. No menu lateral esquerdo clique em Build → Realtime Database
  2. Clique em "Criar banco de dados"
  3. Escolha United States e clique Próxima
  4. Selecione "Iniciar no modo de teste" → Ativar
③ Pegar as credenciais
  1. Clique na engrenagem ⚙ ao lado de Visão geral do projeto
  2. Clique em Configurações do projeto
  3. Role a página até "Seus aplicativos"
  4. Clique no ícone </> (Web)
  5. Digite um apelido (ex: opec) → Registrar app
  6. Aparecerá um bloco de código — copie os valores indicados abaixo
④ Cole as informações aqui
apiKey *
databaseURL * (termina em .firebaseio.com)
projectId *
messagingSenderId
appId *
O plano gratuito do Firebase suporta até 1GB de dados e 10GB de transferência por mês — mais que suficiente para este sistema.
